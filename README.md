# E-commerce Database Analysis (SQL)

This project demonstrates the use of SQL to design and analyze an e-commerce database. It includes creating relational tables, inserting sample data, and performing data analysis using SQL queries.

---

## Features

- Created tables for Customers, Products, and Orders  
- Inserted sample data into each table  
- Performed data analysis using SQL queries  
- Used joins to combine data from multiple tables  
- Applied aggregation functions for insights  

---

## Technologies Used

- SQL (MySQL)
- Relational Database Concepts
- Joins, Aggregations, Group By, Order By

---

## Concepts Implemented

- Table Creation (DDL)
- Data Insertion (DML)
- INNER JOIN
- GROUP BY
- ORDER BY
- Aggregate Functions (SUM, COUNT)
- Filtering using WHERE clause

---

## Key Queries

- Retrieve all customer orders with product details  
- Calculate total number of orders per customer  
- Compute total revenue generated  
- Identify top-selling products  
- Filter customers based on city  
- Retrieve orders after a specific date  

---

## How to Run

1. Open the SQL file in MySQL Workbench or DB Fiddle  
2. Execute the schema (table creation and insert statements)  
3. Run the queries to view results
